Latinx/PLACE User Guide

Here's how to make changes to the code for the Latinx/PLACE web app.

Change Email Address/Phone Number

  • ctrl+f to "If you would like to learn more about...". Highlight the current name and replace with the new one.
  • Highlight the current email address and replace with the new one.

    • Change "About" Content

  • ctrl+f to "Latinx Parents for Change...". This will change the content in the left-side column.
  • ctrl+f to "Working Women Community Centre and The Mennonite...". This will change the content in the right-side column.

    • Add/Remove Carousel Images

    1. First Carousel: ctrl+f to "First Carousel".
      • Photos are in either the "Community Forum" folder.
      • Photos are labelled by number, e.g. image3.jpg, image10.jpg, etc.
      • Select the image to be inserted by copy+pasting two lines of code, starting with the div tag and the img tag immediately following it.
      • Where it says "img/Presentations (or /Community Forum)/image9.jpg, replace the number with the new number of the new image.
    2. Second Carousel: ctrl+f to "Second Carousel"
      • Repeat the same steps as for the first carousel
      • However, make sure to take images from the "Presentations" folder.
    3. Tips
      • If creating a new folder for images, give it a short name so it's easy to type out when calling on the images.
      • Label images plainly so they can be easily used and clearly referred to, e.g. image1.jpg, image2.jpg, image3.jpg
      • To add new images, either batch-upload them to GitHub or copy+paste them to the newly created folder within the "img" folder in the Latinx file you've downloaded from GitHub.

    Add/Remove Modal Content

  • Click open the modal that needs to be changed.
  • Identify the phrase section that needs to be changed and ctrl+f to it in the Sublime code.
  • Add in the new content, save it, then do a hard refresh in the browser to reflect the new content.
  • To ensure the new changes are visible live on the site, upload the newly edited/saved Sublime document to the web hosting service.
